Jogo da Cobrinha

Como Jogar

Use as teclas W, A, S, D para mover a cobra.

Cada vez que comer uma maçã, responda a uma pergunta de lógica proposicional.

Se acertar, a cobra cresce; se errar, a cobra diminui.

O jogo termina se a cobra colidir com si mesma ou se perder todos os segmentos.
